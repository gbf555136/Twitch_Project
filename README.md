## 說明

- 串接[Twitch](https://www.twitch.tv/directory/game/League%20of%20Legends) 的API，顯示英雄聯盟實況列表

- 可切換中文 or 英文實況

- 點擊實況圖片連結到該實況Twitch頁面

- 下拉讀取更多實況台
