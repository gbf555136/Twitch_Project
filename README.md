- $.Ajax 串接 Twitch API

- 手寫`RWD`

- Infinite scroll 讀取更多實況

- 使用`SCSS`、`Jquery` 套件

## 說明

- 模擬Twitch頁面，顯示英雄聯盟實況列表

- 可切換不同語言實況

- 點擊實況圖片連結到該實況Twitch頁面
